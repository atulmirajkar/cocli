# Releases

Pre-built binaries for CoCli.

## Available Binaries

- **cocli-darwin-arm64** - macOS for Apple Silicon (M1, M2, M3)
- **cocli-darwin-amd64** - macOS for Intel
- **cocli-linux-amd64** - Linux x86_64
- **cocli-linux-arm64** - Linux ARM64 (Raspberry Pi, etc.)

## Usage

1. Download the binary for your platform
2. Make it executable: `chmod +x cocli-*`
3. Run: `./cocli-*`

See the main [README.md](../README.md) for complete installation and usage instructions.
